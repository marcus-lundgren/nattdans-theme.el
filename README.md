# nattdans-theme.el
My modified tango-dark theme

To load from a local copy:

```lisp
(use-package nattdans-theme
  :load-path "~/prog/git/my-emacs-theme/")
```

To load from GitHub:

```lisp
(use-package nattdans-theme
  :vc (:url "https://github.com/marcus-lundgren/nattdans-theme.el"
            :rev :newest))
```
